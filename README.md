# CUSTOMER-SERVICE-AGENT-WITH-FEEDBACK-LOOPS

A production-grade, self-improving agentic AI customer service system for NexBank.

## Overview
This repository contains the architecture, design documents, intent taxonomy, guardrails, and continuous learning pipeline for an AI Customer Service Agent.

## Structure
- `docs/`: System documentation (architecture, intent taxonomy, guardrails, etc.)
- `diagrams/`: Architecture diagrams and flowcharts
- `simulations/`: Sample conversation flows
- `tests/`: Guardrail test cases and safety validation
- `config/`: Configuration files and prompt templates
- `CHANGELOG.md`: Daily progress log
